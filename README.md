# Personal Dashboard

A comprehensive personal dashboard with productivity tools, built with HTML, CSS, and JavaScript.

## Features

### Dashboard Tab
- Weather widget for ZIP code 48169
- Real-time clock and calendar
- Quick links (including budget app)
- Top 3 to-dos
- Current goal with progress tracking

### Productivity Tab
- **To-do List**: Add, edit, delete, and mark tasks as complete
- **Goal List**: Set goals with progress percentages, edit and delete
- **Notes Section**: Create dated notes with edit/delete functionality
- **Time Tracker**: Track time spent on tasks
- **Pomodoro Timer**: 25/5/15 minute focus intervals
- **Mood Logger**: Track daily moods with emojis

## Design
- Dark theme with blue color scheme
- Fully responsive for mobile and desktop
- Clean, modern interface
- All data stored locally in browser

## Usage
1. Open `index.html` in a web browser
2. Switch between Dashboard and Productivity tabs
3. Add/edit/delete todos, goals, and notes
4. Use time tracker and Pomodoro timer
5. Log your mood throughout the day

## Data Storage
All data (todos, goals, notes, mood history) is saved in your browser's localStorage. This means:
- Data persists between browser sessions
- Data is specific to your browser/device
- No account or server required
